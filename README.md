
# 线下活动实操手册

一套面向法律行业线下及混合活动的结构化方法资料，覆盖策划、执行、传播、转化与复盘。

本仓库基于《“独一无二”的活动：法律行业市场品牌实战手册》的方法论整理而成，提供可复用的活动框架、参考资料与检索脚本，不包含书稿原文全文。

## 项目内容

- 活动定位与 brief 梳理
- 主题、议程、嘉宾与互动形式设计
- 执行节点、分工、里程碑与风险管理
- 会前、会中、会后传播节奏规划
- 活动后的关系沉淀、转化与复盘方法
- 活动资料的结构化整理与关键词检索

## 适用场景

- 为律所、法律科技公司、法务社群或专业服务机构设计线下活动
- 把一个模糊的活动想法整理成可执行方案
- 优化活动体验、互动形式和记忆点设计
- 建立活动执行清单、传播清单和复盘框架
- 将一套成熟方法论转化为可复用的项目资料库
